## 🛠 روش نصب و اجرای پروژه

### 📥 نصب

1. کلون کردن ریپازیتوری:

```bash
git clone https://github.com/parsamehrpooshan/CryptoApp
```

2. ورود به پوشه پروژه:


```bash
cd project-name
```

3. نصب پکیج‌ها :



```bash
npm install
```


4. اجرای پروژه :


```bash
npm run dev
```

📍 سپس پروژه روی آدرس زیر در دسترس خواهد بود:

```
http://localhost:5173
```
